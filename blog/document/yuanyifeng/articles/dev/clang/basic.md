# 基本语法

## 注释

C语言的注释用`/* .. comments .. */`表示，注释内部可以分行。

有的C编译器还支持双斜杠的`//`的单行注释。

## 变量

变量名可以包括字母、数字和下划线，但是不能以数字开头。

声明变量的时候，需要提供类型。

```c
int num;
num = 42;
```

可以一次声明多个变量。

```c
float x, y, z, X;
```

注意，变量声明的时候，不会清空分配的内存，因此可能会得到任何值。

## 赋值运算符

赋值运算符`=`为变量赋值后，返回等号右边的值。

```c
y = (x = 2 * x)
```

上面代码中，变量`x`先是将自己的值倍增，然后返回这个新的值，赋给变量`y`。
