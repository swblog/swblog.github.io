# PostCSS 的用法

PostCSS是一个使用JavaScript插件转换CSS文件的工具。它允许在CSS文件之中使用JavaScript语法，然后将其转为正常的CSS文件。它可以像普通的CSS预处理器那样工作，但是能够完成多得多的功能。

最著名的基于PostCSS插件就是Autoprefixer。
